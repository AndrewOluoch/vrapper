Visual Mode

Visual mode is still work in progress. It supports character-wise and line-wise selection. While in visual mode, it is not recommended to use the mouse to change the selection. It might work but in general the behaviour is undefined.

You can highlight text using motions and use Vrapper or Eclipse commands on the text. After using an Eclipse command, Vrapper will stay in visual mode, so you have to press Escape to get back to normal mode.

Available commands are:

  • y
  • d
  • c
  • s
  • r
  • x
  • X
  • o
  • <
  • >
  • gc
  • gu
  • gU
  • zz
  • :
